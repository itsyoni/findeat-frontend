# FindEat Business

Restaurant management dashboard for claimed restaurant owners and managers.

## Local development

1. Copy `.env.example` to `.env.local`.
2. Set `VITE_API_URL` to the backend URL and add the Cloudinary cloud name and unsigned upload preset.
3. From the repository root, run `npm run web`.

Build with `npm run build:web`.

Official restaurant post creation intentionally remains in the mobile app.
